Welcome to the IoT Project: Alarm Task

This project demonstrates the use of a microcontroller to control an OLED display and a speaker based on button input.
When the button is pressed, the OLED display will show a message and the speaker will play a sound. When the button is released, the display will clear and the sound will stop.
Since this is an example, we have sped up the code to run every 30 seconds to alert the user.
